# n-crawler
:octocat: 一起学习 node crawler 

## 前沿
这个教程是我很久以前写的关于node.js爬虫的案例，通过整理，重新发布。

在这个系列中，我们将通过抓取[校花网](http://www.xiaohuar.com/list-1-0.html)的妹子图片来学习爬虫。

## 所用到的第三方库
涉及到的库 | 作用
------------ | -------------
request | 请求页面数据
iconv-lite | 处理编码转码
cheerio | 解析页面
async | 异步流程控制
